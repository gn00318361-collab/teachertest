# 青溪國小 115 學年度第 3 次招考第 1 次代理代課教師甄試數位評分控制台

這是一套為青溪國小教師甄試現場量身打造的單頁式「數位評分控制台」與「後台資料庫連動系統」，協助評審委員線上評分、行政端即時進度監控、以及考後成績自動化動態彙整與列印。

---

## 📂 專案檔案結構

* **`index.html`**：網頁控制台主入口（已升級防快取機制，線上版本同步請確保此檔更新）。
* **`today.js`**：核心程式碼，包含 17 位考生資料、試場排程計算、評審防呆機制、鍵盤自動跳格邏輯與新版雲端 API 連結。
* **`today.css`**：介面美化樣式表，支援高質感 Indigo 漸層版面，以及 A4 橫式列印雙分頁強制分頁樣式。
* **`today.html`**：轉址引導檔案。

## 🎯 系統核心功能與防呆優化

### 1. 智慧鍵盤輸入與自動跳下一格（評審端最速輸入）
* **滿 2 位數自動跳格**：評審填寫分數（如 `20`、`18`、`15` 等）一打完第 2 位數字，**游標會瞬間自動跳到下一階輸入框**，完全不需按鍵。個位數分數（如 `8`、`9` 分）可以輸入 `08`、`09` 觸發自動跳格，或直接打單字按 Enter。
* **方向鍵與 Enter 導航**：
  * 按 **`⬇️ 向下鍵`** 或 **`Enter`**：自動往下跳一格。
  * 按 **`⬆️ 向上鍵`**：自動往上回跳一格。
  * 最後一格按 **`Enter`**：自動聚焦至「確認並送出分數」按鈕。
* **聚焦自動全選**：游標點入分數格時會自動全選舊分數，方便直接覆蓋輸入。

### 2. 評審教室防錯防呆機制
* 評審登入（密碼：`csps`）後，若切換到非其負責的流水線考生，系統會**立刻自動鎖定評分面板**並顯示警告訊息（例如：「⚠️ 評審教室與考生流水線不符...」），完全防範不同教室評審誤評的可能。

### 3. 分區獨立監控看板
* 行政看板（3. 行政看板）依據考場地理分區，拆分為兩個精美的獨立進度表格，並**全面放大字體與間距**，方便現場承辦人員即時掌握收卷狀態。

### 4. 🖨️ A4 橫式排程表——強制拆分雙頁列印
行政端點選「列印排程簽名表」（或按 `Ctrl + P`）時，系統會自動將 17 位考生**強制拆分為兩頁 A4 橫式**列印：
* 📄 **第一頁：資優班教室區**（包含 01~09 號考生），附帶承辦人/主任/校長手寫簽章欄。
* 📄 **第二頁：二年級教室區**（包含 10~17 號考生），附帶承辦人/主任/校長手寫簽章欄。
